# Converse

Chrome extension that replaces Twitch's native chat with a custom chat UI. Manifest V3.

## Features

- IRC connection to Twitch chat via WebSocket
- Third-party emotes (7TV, BTTV, FFZ)
- Multiple account support via OAuth
- Resizable chat panel
- Usercards, badges, timestamps, alternating row colors
- Configurable settings (font size, spacing, emote providers, etc.)

## Development

Load the `src/` folder as an unpacked extension in `chrome://extensions` with Developer Mode enabled.
